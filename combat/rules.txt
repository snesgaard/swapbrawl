Start Combat:
  * Draw 4 cards
  * Stamina 0
  * Health max

Start Turn:
  * +1 Stamina
  * Add 3 fixed cards

End turn:
  * Remove fixed cards from hand

Draw Card:
  * Add to hand
  * If hand full, discard oldest

Ailment:
  * Always 1 damage
  * Above threshold triggers ailment
  * Ailment trigger doubles next threshold
